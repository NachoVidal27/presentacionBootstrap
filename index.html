Introducción al UI

¿Qué es el diseño UI?

La expresión UI viene de User Interface, que significa Interfaz del Usuario. La idea del UI es guiar al usuario por tu aplicación durante el tiempo que él la usa.

Utilizaremos el mismo ejemplo que acabamos de ver para facilitar la explicación.

En el momento que una persona entra en tu sitio web, existen algunas acciones específicas que deseas que ellas tomen.

3 diferencias fundamentales entre diseño UI y UX

UI solo existe con un interfaz visual, UX no

UI direcciona al usuario, UX hace la navegación valer la pena

UI es sobre razón, UX es sobre emoción

Cuales son los tipos de interfaz de UI?

Interfaz gráfica de usuario (GUI) , utiliza elementos como ventanas, iconos y menús para fomentar la interacción

Interfaz de linea de comandos (CLI) permite a los usuarios interactuar con el sistema usando comandos de texto

Interfaz de usuario basada en gestos recurre a gestios manuales como los que se realizan en pantallas táctiles.

Interacción mediante comandos de voz como la que se emplea en asistentes como Siri o Alexa

Que compone la UI?

Elementos visuales botones, menús o iconós que facilitan la interacción.

Diseño de la navegación , es la estructura que permite a los usuarios mvoerse facilmente por el sistema o aplicación.

Layouts y grids , permiten la organización visual, facilitando la colocación de los elementos en la pantalla.

Tipografías y colores , que mejoran la legibilidad y la estética del diseño.

Características de una buena UI

La UI debe permitir a los usuarios comprender y utilizar la inferzar sin dificultad

Debe ser usable en diferentes dispositivos, desde computadoras hasta celulares

Deberá ser visualmente atractiva y mantener una coherencia en el diseño

Bootstrap

Es un framework front-end de código abierto que facilita el diseño y desarrollo de sitios web y aplicaciones responsivas y móviles.

Desarrollado originalmente dentro de Twitter, con el objetivo de crear una herramienta para mantener la coherencia de los diseños dentro de la compañía.

Fue lanzado en 2011

Bootstrap sirve para diseñar sitios web modernos y responsivos más rápido, usando componentes y estilos listos para usar.

Mantener consistencia visual en proyectos grandes o colaborativos, usando una base de diseño común.

Para prototipar interfaces rápidamente, ideal en etapas tempranas de desarrollo.

Permite aplicar estilos directamente desde el HTML usando clases predefinidas, sin necesidad de escribir CSS personalizado.

Sistema de grillas: Bootstrap incorpora un sistema de rejilla flexible y responsivo que permite organizar el contenido de forma estructurada a través de filas y columnas. Este sistema facilita la creación de layouts que se adaptan automáticamente a diferentes tamaños de pantalla

Componentes predefinidos (HTML + CSS): Incluye una amplia variedad de componentes listos para usar (Botones, Formularios, Alertas)

Integración con JavaScript: Incluye scripts que permiten funcionalidades como Menús desplegables, Carruseles de imágenes, Modales
